#### **[Mockup #1] Problem Definition: Gap 발견 슬라이드**
*   **목적:** 청중(40-50대 CEO)에게 현재 시장 데이터의 *불완전함*과 그로 인한 '잠재적 손실'을 시각적으로 체감하게 합니다. (데이터가 문제임을 깨닫게 함).
*   **레이아웃/구조:** 3분할 구조. 좌측에 핵심 지표, 중앙에 Gap 시각화, 우측에 간결한 위기 메시지 배치.

| 요소 | 상세 디자인 가이드라인 | 데이터 적용 규칙 |
| :--- | :--- | :--- |
| **제목** | "MEA 시장, 당신의 비즈니스 성장을 저해하는 3가지 Blind Spot" (딥 사파이어 블루) | - |
| **핵심 지표 시각화** | 막대 그래프 형태. 목표치(Goal)는 파란색으로 명확히 제시하고, 실제 데이터(Actual)는 얇은 회색 라인으로 보여줍니다. Gap 부분에만 Alert Orange 색상을 사용합니다. | $ \text{Gap} = (1 - \frac{\text{Actual}}{\text{Goal}}) \times 100\% $ 에 해당하는 비율을 오렌지 그라데이션으로 채워야 합니다. |
| **주요 설명 (좌측)** | '데이터 파편화', '시장 변동성 측정 불가', '지역별 비효율적 자원 배분' 등 키워드를 아이콘과 함께 제시합니다. | 모든 지표 근거는 출처(Source)를 작은 글씨로 반드시 명시하여 신뢰도를 높입니다. |
| **톤앤매너** | 충격적이면서도 데이터 기반의 냉정한 경고 톤. |

#### **[Mockup #2] Solution Mechanism: Trust Score 시각화 슬라이드 (★최핵심)**
*   **목적:** MEA SalesCompass가 어떻게 '불확실성'을 측정하고, 이를 구매 결정에 필수적인 '신뢰도 점수(Trust Score)'로 변환하는지 기술적으로 설득합니다. (기술력을 통한 신뢰 확보).
*   **레이아웃/구조:** 다이어그램 중심의 인과관계 흐름도.

| 요소 | 상세 디자인 가이드라인 | 데이터 적용 규칙 |
| :--- | :--- | :--- |
| **제목** | "리스크를 기회로: MEA SalesCompass의 Trust Score 엔진" (딥 사파이어 블루) | - |
| **핵심 시각화** | 3단계 플로우 다이어그램 구조. `Raw Data` $\to$ `Integrity Middleware` $\to$ `Trust Score`. 화살표는 파란색 계열을 사용합니다. | 이 과정에서 데이터가 불완전할수록, 또는 검증 단계(Middleware)를 거치지 않을수록 흐름의 폭이 넓고 **오렌지 그라데이션**으로 채워져야 합니다. |
| **Trust Score 표현** | 중앙에 원형 게이지(Radial Gauge) 형태를 사용합니다. 점수 구간별로 색상을 명확히 정의합니다: (90-100점: Deep Blue), (70-89점: Sapphire), (50-69점: Warning Yellow), (<50점: Alert Orange). | 이 게이지는 단순 수치 표시가 아닌, **위험 범위의 반경**으로 표현되어야 합니다. |
| **텍스트 강조** | "불확실성은 비용입니다."라는 문구를 슬라이드 하단에 배치하여 메시지를 각인시킵니다. |

#### **[Mockup #3] Value Proposition: Tiered Pricing & ROI 연결 슬라이드**
*   **목적:** 데이터 신뢰도(Trust Score)가 곧 '투자 대비 수익률(ROI)'임을 논리적으로 완성하고, 자연스럽게 상품 구매로 연결합니다. (구매 동기 부여).
*   **레이아웃/구조:** 비교형 테이블 및 ROI 그래프 조합.

| 요소 | 상세 디자인 가이드라인 | 데이터 적용 규칙 |
| :--- | :--- | :--- |
| **제목** | "최적의 파트너십: 신뢰 기반 성장을 위한 3단계 솔루션" (딥 사파이어 블루) | - |
| **가격 구조 시각화** | Bronze, Silver, Gold 세 가지 티어를 가로축으로 나열하는 비교 테이블. 각 기능 항목(데이터 접근 범위, Trust Score 깊이, 컨설팅 지원 등)을 체크박스 또는 아이콘으로 표현합니다. | 'Gold' 티어의 핵심 차별점은 **"실시간 불확실성 예측 모델 (Advanced Warning System)"**이며, 이는 오직 최고 수준의 데이터 신뢰도(Trust Score 90+)를 전제로 함을 강조해야 합니다. |
| **ROI 연결 그래프** | X축: 투자 금액 (Bronze $\to$ Gold). Y축: 예상 수익률 상승폭. 가격이 높아질수록 곡선이 가파르게 상승하는 형태로 그립니다. | 이 그래프의 기울기는 데이터 신뢰도와 정비례함을 강조해야 합니다. |

**작성 목적:** MEA SalesCompass의 비즈니스 가치(데이터 신뢰도 기반 리스크 관리)를 CEO급 청중에게 가장 효과적으로 전달하는 3대 핵심 슬라이드 시각 시스템 확정.

## 🎨 전역 스타일 및 색상 정의
*   **메인 컬러 (신뢰):** Deep Sapphire Blue (#0A2342) - 제목, 주요 데이터, 배경 강조색.
*   **경고/문제점:** Alert Orange (#D9534F) - Gap, 리스크 범위, 경고 메시지.
*   **보조 컬러 (중립):** Light Gray (#E0E6ED), Navy Blue Gradient.
*   **폰트:** 전문적인 느낌의 고딕 계열 폰트 조합.

## 🖼️ [Mockup #1] Problem Definition: Gap 발견 슬라이드 명세
### 목표 메시지: "당신이 모르는 리스크가 문제입니다."
1.  **구조:** 'Metric Name' (좌) $\to$ 'Current Data Status' (중앙, 시각화) $\to$ 'Impact/Action Needed' (우).
2.  **시각화 상세:** 막대 그래프 사용. 목표치와 실제 데이터의 차이(Gap)를 오렌지 그라데이션으로 채운다. 이 Gap 부분은 단순한 부족함이 아니라 **'측정되지 않은 불확실성'**을 의미하도록 명시한다.
3.  **CTA:** 슬라이드 하단에 "데이터 기반 리스크 진단을 시작하십시오." 라는 문구를 배치하여 다음 슬라이드로의 흐름을 유도한다.

## 📊 [Mockup #2] Solution Mechanism: Trust Score 시각화 슬라이드 명세 (핵심)
### 목표 메시지: "우리는 불확실성을 측정하고, 이를 점수로 확신합니다."
1.  **구조:** 인과관계 다이어그램 (Flow Chart). **[Raw Data Input] $\to$ [Data Integrity Middleware] $\to$ [Trust Score Output].**
2.  **Trust Score 게이지:** 원형/반원형 그래프를 사용하며, 점수가 낮을수록 오렌지 그라데이션이 더 넓게 채워진다. (예: 60점일 경우, 전체 범위의 40%가 Alert Orange로 표시).
3.  **텍스트 강조:** '불확실성 비용 측정'이라는 문구를 타이포그래피적으로 가장 크게 배치하여, 우리가 판매하는 것이 단순 데이터가 아니라 **'비용 예측 능력'**임을 각인시킨다.

## 📈 [Mockup #3] Value Proposition: Tiered Pricing & ROI 연결 슬라이드 명세
### 목표 메시지: "신뢰도가 높을수록 투자 대비 수익률은 기하급수적으로 높아집니다."
1.  **구조:** 비교 테이블과 그래프의 결합.
2.  **가격 테이블:** Bronze, Silver, Gold 3단계로 구성하며, 핵심 기능 옆에 Trust Score 수준(예: Basic -/Medium -/High)을 명시하여 가격이 **기술적 깊이**와 연결됨을 보여준다.
3.  **ROI 그래프:** X축 (투자액), Y축 (성장률). Gold 티어는 단순 선형 증가가 아닌, 가파른 곡선(Exponential Growth Curve)으로 표현하여 기대 효과의 극대화를 시각적으로 설득한다.
