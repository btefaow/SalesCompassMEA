## 🎯 목적 및 적용 범위
*   **목적:** 발표 자료(Pitch Deck, Infographic)의 논리를 유지하면서, 장문의 전문 컨설팅 보고서(Report/Booklet)에 필요한 구조적 통일성과 신뢰성을 부여합니다.
*   **타겟 청중:** 40~50대 고위 의사결정권자 (CEO, 대표이사). 즉각적인 지식 전달보다는 '깊은 고민'을 유도해야 합니다.
*   **톤앤매너:** 학술적이고, 데이터 기반이며, 권위적임.

## 🎨 핵심 디자인 요소 정의

### 1. 컬러 시스템 (Color Palette)
| 용도 | 색상 코드 (HEX) | 역할 및 사용 규칙 |
| :--- | :--- | :--- |
| **메인 배경/텍스트** | #2D3748 (Dark Slate Gray) / #333333 | 본문 텍스트, 제목. 높은 가독성을 위해 순수한 블랙 대신 톤 다운된 차콜 사용. |
| **주요 강조색 (Primary)** | Deep Sapphire Blue (#0A2342) | 섹션 헤더, 장 제목(Chapter Title), 주요 그래프 배경, 회사의 정체성 색상으로 활용. (신뢰/전문성 상징) |
| **경고/Gap 강조색 (Secondary)** | Alert Orange (#D9534F) | 문제점(Problem), Gap 발견 지표, 리스크 경고 표시 등 '주의'가 필요한 데이터에 한하여 사용. (즉각적 관심 유도) |
| **중립/보조색** | #AAB7B8 (Soft Gray) | 구분선, 부차적인 설명 텍스트, 그래프의 배경 음영 처리 등에 사용.

### 2. 타이포그래피 시스템 (Typography)
*   **헤드라인 (H1 - 장 제목):** Pretendard Bold / 24pt ~ 36pt. (Deep Sapphire Blue). 강력하고 명확하게 섹션을 분리.
*   **소제목 (H2 - 섹션 제목):** Pretendard SemiBold / 18pt ~ 24pt. (Dark Slate Gray).
*   **본문 텍스트 (Body Text):** Pretendard Regular / 10pt ~ 12pt. (Dark Slate Gray). 충분한 줄 간격(Line-height: 1.6)을 확보하여 가독성 극대화.
*   **캡션/참고 자료:** Pretendard Light / 8pt ~ 10pt. (Soft Gray).

| **[Key Takeaway Box]** | 청중이 보고서 전체를 읽지 않아도 핵심 메시지를 즉각적으로 파악하게 함. | 회색 박스 배경(#F7FAFC), 좌측 상단에 'KEY INSIGHT' 라벨링 (Alert Orange). 본문 텍스트 대비 약간 더 진한 배경을 사용하여 분리감 부여. | 반드시 장의 끝이나 섹션의 요약 부분에 배치하여, 논리적 결론을 시각적으로 확정하는 역할을 수행해야 함. |
