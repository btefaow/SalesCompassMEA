# 🤹 Advisor 스킬 (Skills)

이 폴더는 Advisor 에이전트가 활용할 수 있는 프롬프트 스킬과 도구(Tools)를 보관하는 곳입니다.

## 기본 장착 스킬
*   **고객사 빙의 (Roleplay)**: 타깃 국가/기업의 정보를 입력받아 해당 업체의 보수적인 CEO로 완벽히 빙의하여 질문을 던지는 스킬.
*   **논리 검증 (Logic Audit)**: 현재 작성된 영업 스크립트(Playbook)에서 '수치적 근거가 부족한 부분'을 찾아내어 공격하는 스킬.
*   **반론 극복 생성기 (Objection Handling Generator)**: 예상되는 거절 멘트에 대해 "공감 $\rightarrow$ 논리적 반박(데이터) $\rightarrow$ 재제안" 구조의 스크립트를 즉시 생성하는 스킬.
